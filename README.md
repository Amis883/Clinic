# Clinic Website

A simple responsive clinic website built with HTML, CSS, and JavaScript.
The project is bundled using Parcel and deployed on Vercel.

## 🚀 Live Demo

(https://clinic-2sl0-9u7gfpzrf-amis883s-projects.vercel.app/)

## 📌 Features

* Responsive layout
* Modern UI design
* Contact and information sections
* Fast build using Parcel bundler
* Deployed with Vercel

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
* Parcel
* Vercel

```
project
```

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/amis883/clinic.git
```

Install dependencies:

```
npm install
```

Run the project locally:

```
npm start
```

Build the project:

```
npm run build
```

## 🌐 Deployment

The project is deployed using **Vercel**.
Build output directory:

```
dist
```
